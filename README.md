# Desarrollo de Videojuegos en Java

Repositorio donde registro mi proceso de aprendizaje, prácticas y ejercicios sobre desarrollo de videojuegos 2D utilizando Java.

## Acerca de este repositorio

Este repositorio está pensado para organizar y documentar mi aprendizaje en la creación de videojuegos en Java. Aquí guardaré código, notas y explicaciones relacionadas: manejo de pantallas, tiles, renderizado, lógica del juego y otros fundamentos importantes.

Aunque este material está orientado a mi uso personal, incluyo las referencias de los tutoriales y recursos que sigo para que cualquier persona pueda consultarlos directamente.

Además, todas las explicaciones detalladas y el avance paso por paso estarán organizados en mi Notion:

## Estructura del repositorio

* `src/` — Código fuente organizado por módulos
* `recursos/` — Sprites, imágenes de prueba u otros elementos gráficos.

## Notas finales

* El repositorio se irá actualizando conforme avance en el curso.
* El objetivo es comprender los fundamentos reales de un juego 2D, no solo copiar el código.
* Si quieres seguir el mismo camino de aprendizaje, te recomiendo revisar directamente la playlist mencionada y la documentación que iré agregando en Notion.
